About Survey Tools

This is meant to be a online resource that can be used by surveyors in the field to make simple computations.

It has the following abilities:

  1. Compute area based on coordinate inputs

It still under active development with the intention of adding the following features:

  1. Provide map interface for input of coordinates
  2. Create api endpoints
  3. Develop and link to an android app
  4. Integrate the user model
  5. Allow user to save edits
  6. Share land specs on social media
